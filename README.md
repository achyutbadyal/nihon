## Features (Inherited from Mihon)

- Local reading of content.
- A configurable reader with multiple viewers and reading directions.
- Tracker support (MyAnimeList, AniList, Kitsu, MangaUpdates, Shikimori, Bangumi).
- Categories to organize your library.
- Light and dark themes.
- Scheduled library updates.
- Local backups.

## Features (Added)

- Parallel downloading of chapters (per source)
- Customizable page/chapter preload settings.
- Chapter list in reader
